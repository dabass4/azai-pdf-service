# AZAI - Healthcare Timesheet Management Platform

## Application Name
**AZAI** (formerly Healthcare Timesheet Scanner)

## Description
AZAI is a comprehensive multi-tenant healthcare timesheet management and Medicaid claims processing platform designed for Ohio healthcare providers.

## Key Features
- AI-powered timesheet scanning and processing
- Multi-tenant architecture for managing multiple client organizations
- Ohio Medicaid (OMES) EDI integration for claims submission
- Availity clearinghouse integration
- Real-time patient eligibility verification
- Claims tracking and status monitoring
- Admin panel for software providers
- Secure credential management per organization
